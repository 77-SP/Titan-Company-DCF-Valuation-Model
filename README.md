# Titan Company Ltd. – DCF Valuation Model (Microsoft Excel)

## Overview

This project presents a Discounted Cash Flow (DCF) valuation model for Titan Company Ltd., built entirely in Microsoft Excel.

The model estimates the intrinsic value of the company by forecasting future Free Cash Flow to Firm (FCFF) and discounting it using the Weighted Average Cost of Capital (WACC). The workbook is fully linked across multiple worksheets to ensure that updates to assumptions automatically flow through the valuation.
