Titan Company Ltd. – DCF Valuation Model (Microsoft Excel)
Overview

This project presents a Discounted Cash Flow (DCF) valuation model for Titan Company Ltd. built entirely in Microsoft Excel. The model estimates the intrinsic value of the company by forecasting future cash flows and discounting them using the Weighted Average Cost of Capital (WACC).

The workbook was developed by linking multiple worksheets to create a structured financial model that follows a standard valuation process.

Key Features: 

Historical financial statement analysis
Revenue and EBIT forecasting
FCFF calculation
WACC estimation
Terminal value calculation
Enterprise value estimation
Equity value estimation
Sensitivity analysis
Financial dashboard summarizing key valuation outputs
